# 06. 문제 유형과 손실 함수

## 이번 장에서 배울 것

딥러닝 모델을 만들 때 가장 먼저 해야 하는 일은 문제 유형을 정하는 것이다.

```text
숫자를 예측하는가?
예/아니오를 고르는가?
여러 종류 중 하나를 고르는가?
```

문제 유형이 달라지면 출력층과 손실 함수도 달라진다.

## 세 가지 기본 문제 유형

입문 단계에서는 먼저 세 가지를 구분하면 된다.

```text
회귀: 숫자 하나를 예측
이진 분류: 둘 중 하나를 예측
다중 분류: 여러 종류 중 하나를 예측
```

## 회귀

회귀는 숫자를 예측하는 문제다.

예:

```text
내일 기온 예측
집값 예측
자전거 대여량 예측
미세먼지 수치 예측
```

출력층은 보통 다음처럼 만든다.

```python
Dense(1)
```

출력값이 숫자 하나이기 때문이다.

손실 함수는 보통 `mse`를 사용한다.

```python
loss="mse"
```

## 이진 분류

이진 분류는 둘 중 하나를 고르는 문제다.

예:

```text
스팸 / 정상
당뇨 위험 있음 / 없음
심장 질환 있음 / 없음
퇴사함 / 퇴사하지 않음
```

출력층은 보통 다음처럼 만든다.

```python
Dense(1, activation="sigmoid")
```

sigmoid는 출력값을 0과 1 사이로 바꿔준다. 그래서 확률처럼 해석할 수 있다.

손실 함수는 보통 `binary_crossentropy`를 사용한다.

```python
loss="binary_crossentropy"
```

## 다중 분류

다중 분류는 여러 종류 중 하나를 고르는 문제다.

예:

```text
숫자 0~9 중 하나 분류
꽃 5종류 중 하나 분류
동물 종류 분류
```

클래스가 5개라면 출력층은 보통 다음처럼 만든다.

```python
Dense(5, activation="softmax")
```

softmax는 각 클래스의 점수를 확률처럼 바꾸고, 전체 합이 1이 되게 만든다.

손실 함수는 라벨 형태에 따라 달라진다.

정답이 정수 라벨이면:

```python
loss="sparse_categorical_crossentropy"
```

정답이 원핫 인코딩이면:

```python
loss="categorical_crossentropy"
```

## 정수 라벨과 원핫 라벨

꽃 종류가 5개 있다고 하자.

정수 라벨은 이렇게 표현한다.

```text
roses = 2
```

원핫 라벨은 이렇게 표현한다.

```text
roses = [0, 0, 1, 0, 0]
```

둘 다 같은 뜻이지만 모양이 다르다. 그래서 손실 함수도 맞춰야 한다.

## 기본 선택표

처음에는 이 표를 외우기보다 자주 보면서 익히면 된다.

```text
회귀:
출력층 Dense(1)
손실 mse

이진 분류:

손실 binary_crossentropy

다중 분류, 정수 라벨:
출력층 Dense(class_count, activation="softmax")
손실 sparse_categorical_crossentropy

다중 분류, 원핫 라벨:
출력층 Dense(class_count, activation="softmax")
손실 categorical_crossentropy
```
