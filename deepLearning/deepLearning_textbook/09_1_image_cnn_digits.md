# 09. 이미지 분류와 CNN 입문

## 이번 장에서 배울 것

이번 장에서는 이미지 데이터를 딥러닝으로 분류하는 기본 흐름을 배운다.

모델은 CNN을 사용한다.

```text
CNN = Convolutional Neural Network = 합성곱 신경망
```

이번에는 큰 이미지가 아니라 작은 손글씨 숫자 이미지를 사용한다.

```text
0부터 9까지 숫자 이미지 분류
```

## 왜 이미지에는 CNN을 쓸까

이미지는 픽셀들의 모음이다.

숫자 3 이미지를 사람은 쉽게 알아보지만, 컴퓨터에게는 작은 숫자들이 격자로 놓인 배열이다.

```text
밝은 픽셀
어두운 픽셀
선의 방향
곡선
모서리
```

CNN은 작은 필터를 움직이며 이미지 안의 특징을 찾는다.

비유하면 작은 돋보기를 들고 이미지를 훑는 것과 비슷하다.

## 이미지 데이터의 모양

딥러닝에서 이미지는 보통 다음 모양으로 들어간다.

```text
(샘플 수, 높이, 너비, 채널 수)
```

이번 데이터는 흑백 숫자 이미지다.

```text
높이: 8
너비: 8
채널 수: 1
```

흑백 이미지는 채널이 1개다. 컬러 이미지는 보통 RGB라서 채널이 3개다.

## CNN의 기본 층

처음에는 네 가지를 기억하면 된다.

```text
Conv2D: 이미지에서 특징을 찾는다.
MaxPooling2D: 중요한 특징만 남기고 크기를 줄인다.
Flatten: 2차원 이미지 특징을 1차원으로 펼친다.
Dense: 최종 분류를 한다.
```

## 이번 문제의 유형

숫자 0부터 9까지 10개 중 하나를 고르는 문제다.

따라서 다중 분류다.

출력층은 다음처럼 만든다.

```python
Dense(10, activation="softmax")
```

정답 라벨은 0, 1, 2 같은 정수이므로 손실 함수는 다음을 사용한다.

```python
loss="sparse_categorical_crossentropy"
```

## 이번 장에서 반드시 기억할 문장

```text
CNN은 이미지에서 작은 특징을 찾아내고, 그 특징들을 모아 최종 분류를 한다.
```

## 확인 질문

1. 흑백 이미지의 채널 수는 몇 개인가?
2. 컬러 이미지의 채널 수는 보통 몇 개인가?
3. `Conv2D`는 어떤 역할을 하는가?
4. `Flatten`은 왜 필요한가?
5. 숫자 0~9 분류는 회귀, 이진 분류, 다중 분류 중 무엇인가?

## 과제

1. CNN의 네 가지 기본 층 역할을 한 문장씩 써라.
2. `filters=16`을 `filters=32`로 바꾸면 무엇이 달라질지 추측해보라.
3. epoch 수를 5에서 10으로 늘려보고 검증 정확도가 어떻게 달라지는지 확인해보라.
