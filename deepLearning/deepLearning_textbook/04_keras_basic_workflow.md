# 04. Keras 기본 워크플로우

## 이번 장에서 배울 것

이번 장에서는 Keras로 딥러닝 모델을 만들 때 반복해서 등장하는 기본 흐름을 배운다.

```text
모델 만들기 -> 컴파일하기 -> 학습하기 -> 평가하기 -> 예측하기
```

이 흐름은 앞으로 CNN, LSTM, GRU를 배울 때도 계속 반복된다.

## Keras란 무엇인가

Keras는 딥러닝 모델을 쉽게 만들 수 있게 도와주는 라이브러리다.

딥러닝 모델을 직접 밑바닥부터 만들려면 많은 계산을 직접 처리해야 한다. Keras는 그 복잡한 부분을 감싸고, 우리가 모델 구조와 학습 방법에 집중할 수 있게 해준다.

이번 교재에서는 현재 환경에 맞춰 독립 Keras 3 스타일을 사용한다.

```python
from keras.models import Sequential
from keras.layers import Dense, Input
from keras.optimizers import SGD
```

## Keras 모델의 5단계

Keras로 모델을 만들 때는 보통 다음 순서를 따른다.

```text
1. 데이터 준비
2. 모델 만들기
3. 컴파일하기
4. 학습하기
5. 예측하기 또는 평가하기
```

각 단계는 역할이 다르다.

## 1단계. 데이터 준비

모델에게는 입력과 정답이 필요하다.

```python
x_data = ...
y_data = ...
```

여기서 `x_data`는 모델이 보는 입력이고, `y_data`는 모델이 맞혀야 하는 정답이다.

Keras 모델은 보통 입력을 2차원 형태로 받는다.

```text
(샘플 수, 특성 수)
```

예를 들어 입력값이 5개이고, 각 입력에 특성이 1개뿐이라면 모양은 다음과 같다.

```text
(5, 1)
```

## 2단계. 모델 만들기

가장 단순한 모델은 다음처럼 만든다.

```python
model = Sequential([
    Input(shape=(1,)),
    Dense(1)
])
```

각 부분의 뜻은 다음과 같다.

```text
Sequential: 층을 순서대로 쌓는 모델
Input(shape=(1,)): 입력값이 1개라는 뜻
Dense(1): 출력값이 1개라는 뜻
```

`Dense` 층 안에는 모델이 학습할 가중치와 편향이 들어 있다.

## 3단계. 컴파일하기

모델을 만들었다고 바로 학습할 수 있는 것은 아니다. 학습 전에 모델에게 두 가지를 알려줘야 한다.

```text
1. 얼마나 틀렸는지 어떻게 계산할 것인가
2. 틀린 것을 어떤 방식으로 고칠 것인가
```

이 설정을 하는 메서드가 `compile()`이다.

```python
model.compile(
    optimizer=SGD(learning_rate=0.01),
    loss="mse"
)
```

여기서 `loss="mse"`는 평균 제곱 오차로 손실을 계산하겠다는 뜻이다.

`optimizer`는 손실을 줄이기 위해 모델 내부 숫자를 어떻게 수정할지 정하는 방법이다.

## 4단계. 학습하기

학습은 `fit()`으로 한다.

```python
history = model.fit(
    x_data,
    y_data,
    epochs=200,
    verbose=0
)
```

`fit()`은 다음 뜻이다.

```text
입력과 정답을 보고, 손실이 줄어들도록 모델 내부 숫자를 고친다.
```

`epochs`는 전체 데이터를 몇 번 반복해서 볼지 정한다.

`history`에는 학습 중 손실값 변화가 저장된다.

## 5단계. 예측하기

학습된 모델로 새로운 값을 예측할 때는 `predict()`를 사용한다.

```python
model.predict(new_x)
```

주의할 점은 새 입력도 학습 때와 같은 형태로 넣어야 한다는 것이다.

예를 들어 모델이 `(샘플 수, 특성 수)` 형태를 기대한다면, `x=5` 하나를 예측할 때도 다음처럼 2차원으로 넣는다.

```python
new_x = np.array([[5]], dtype=np.float32)
```

## 자주 쓰는 메서드 정리

```text
summary(): 모델 구조를 확인한다.
compile(): 학습 방법을 설정한다.
fit(): 모델을 학습시킨다.
predict(): 새 입력에 대해 예측한다.
get_weights(): 모델이 학습한 내부 숫자를 확인한다.
```

## 이번 장에서 반드시 기억할 문장

```text
Keras의 기본 흐름은 모델 만들기, 컴파일하기, 학습하기, 예측하기다.
```

그리고 이것도 기억하자.

```text
compile은 학습 준비, fit은 실제 학습, predict는 예측이다.
```
