# 03. 손실과 학습

## 이번 장에서 배울 것

이번 장에서는 모델이 얼마나 틀렸는지 숫자로 표현하는 방법을 배운다.

핵심 단어는 `손실`이다.

```text
손실(loss) = 모델이 얼마나 틀렸는지 나타내는 숫자
```

딥러닝 학습은 손실을 줄이는 방향으로 모델 내부 숫자를 고치는 과정이다.

## 손실을 왜 계산해야 할까

모델이 예측을 했다고 하자.

```text
정답: 9
예측: 6
```

사람은 바로 알 수 있다.

```text
3만큼 틀렸다.
```

하지만 모델을 학습시키려면 “틀렸다”를 숫자로 계산해야 한다. 그래야 얼마나 고쳐야 할지 판단할 수 있다.

## 오차와 손실

먼저 오차를 보자.

```text
오차 = 정답 - 예측
```

예를 들어 정답이 9이고 예측이 6이면:

```text
오차 = 9 - 6
오차 = 3
```

예측이 정답보다 크면 오차가 음수가 될 수도 있다.

```text
정답: 9
예측: 12

오차 = 9 - 12
오차 = -3
```

그런데 모델이 얼마나 틀렸는지 볼 때는 방향보다 크기가 더 중요할 때가 많다.

```text
3만큼 작게 틀림
3만큼 크게 틀림
```

둘 다 “3만큼 틀림”이다.

그래서 회귀 문제에서는 오차를 제곱해서 양수로 만든 뒤 평균을 내는 방식을 많이 쓴다.

## MSE란 무엇인가

MSE는 Mean Squared Error의 줄임말이다.

한국어로는 평균 제곱 오차라고 한다.

```text
Mean = 평균
Squared = 제곱한
Error = 오차
```

즉 MSE는 다음 뜻이다.

```text
오차를 제곱하고, 그 값들의 평균을 낸 것
```

수식처럼 보이면 어렵지만, 말로 풀면 이렇다.

```text
1. 정답과 예측의 차이를 구한다.
2. 차이를 제곱한다.
3. 모든 데이터에 대해 평균을 낸다.
```

## 왜 제곱을 할까

제곱을 하면 두 가지 효과가 있다.

첫째, 음수가 양수가 된다.

```text
(-3) * (-3) = 9
```

둘째, 크게 틀린 값을 더 크게 벌준다.

```text
1만큼 틀림 -> 제곱하면 1
3만큼 틀림 -> 제곱하면 9
10만큼 틀림 -> 제곱하면 100
```

그래서 MSE는 크게 틀린 예측에 민감하다.

## 학습이란 무엇인가

모델은 예측하고, 손실을 계산하고, 손실이 줄어드는 방향으로 내부 숫자를 바꾼다.

```text
예측
손실 계산
w와 b 수정
다시 예측
다시 손실 계산
다시 수정
```

이것이 학습이다.

## 비유: 다트 던지기

모델 학습은 다트 던지기와 비슷하다.

처음에는 중심에서 멀리 떨어진 곳에 다트를 던질 수 있다.

```text
중심 = 정답
다트가 꽂힌 위치 = 예측
중심까지 거리 = 손실
```

다음번에는 조금 더 중심에 가깝게 던지려고 자세를 고친다.

모델도 마찬가지다.

```text
정답과 예측의 차이를 보고,
내부 숫자인 w와 b를 조금씩 고친다.
```

## 코드에서 손실은 어디에 있는가

Keras 코드에서는 `compile()`에서 손실 함수를 정한다.

```python
model.compile(
    optimizer=SGD(learning_rate=0.01),
    loss="mse"
)
```

여기서 `loss="mse"`는 다음 뜻이다.

```text
예측이 얼마나 틀렸는지 MSE 방식으로 계산하겠다.
```

그리고 `fit()`을 실행하면 모델은 이 손실을 줄이려고 노력한다.

```python
model.fit(x_data, y_data, epochs=200)
```

## 손실 그래프 읽기

학습이 잘 되고 있다면 손실 그래프는 대체로 내려간다.

```text
손실이 내려감 = 모델이 덜 틀리게 되고 있음
손실이 그대로임 = 모델이 잘 배우지 못하고 있음
손실이 커짐 = 학습 설정이나 데이터에 문제가 있을 수 있음
```

처음에는 그래프가 완벽히 매끄럽지 않아도 괜찮다. 중요한 것은 전체적으로 줄어드는 흐름이다.

