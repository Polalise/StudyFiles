# 02. 아주 작은 수식 읽기

## 이번 장에서 배울 것

이번 장의 목표는 수식을 잘 푸는 것이 아니다. 수식을 보고 겁먹지 않고, 문장처럼 읽는 연습을 하는 것이다.

딥러닝에서 수식은 자주 나오지만, 처음부터 복잡하게 이해할 필요는 없다. 우리는 먼저 아주 작은 수식부터 읽는다.

```text
y = wx + b
```

이 수식을 읽을 수 있으면, 딥러닝의 가장 기본적인 생각을 이해할 수 있다.

## 수식은 압축된 문장이다

수식은 어려운 암호가 아니다. 긴 말을 짧게 줄인 문장이다.

예를 들어 다음 문장을 보자.

```text
입력값 x에 2를 곱하고 1을 더하면 y가 된다.
```

이 문장을 짧게 쓰면 다음과 같다.

```text
y = 2x + 1
```

그러니까 수식을 볼 때는 이렇게 생각하면 된다.

```text
이 수식은 어떤 문장을 짧게 적은 것일까?
```

## y = wx + b 읽기

가장 기본 수식은 다음과 같다.

```text
y = wx + b
```

읽는 순서는 오른쪽에서 왼쪽으로 가는 편이 쉽다.

```text
wx + b를 계산하면 y가 된다.
```

조금 더 풀어 쓰면 다음과 같다.

```text
입력값 x에 가중치 w를 곱하고,
편향 b를 더하면,
예측값 y가 된다.
```

기호별 뜻은 다음과 같다.

```text
x: 입력값
w: 가중치
b: 편향
y: 예측값
```

## w는 무엇인가

`w`는 weight의 첫 글자다. 한국어로는 가중치라고 한다.

가중치는 입력값이 결과에 얼마나 강하게 영향을 주는지 정하는 숫자다.

예를 들어 다음 두 규칙을 비교해보자.

```text
y = 1x + 0
y = 5x + 0
```

같은 `x`를 넣어도 `w`가 5이면 결과가 훨씬 크게 변한다.

```text
x = 2

y = 1x + 0 -> 2
y = 5x + 0 -> 10
```

즉 `w`는 입력값에 주는 힘이라고 볼 수 있다.

## b는 무엇인가

`b`는 bias의 첫 글자다. 한국어로는 편향이라고 한다.

편향은 입력값과 상관없이 기본으로 더해지는 숫자다.

예를 들어 다음 두 규칙을 비교해보자.

```text
y = 2x + 0
y = 2x + 3
```

둘 다 `x`에 2를 곱하지만, 두 번째 규칙은 항상 3만큼 더 크다.

```text
x = 4

y = 2x + 0 -> 8
y = 2x + 3 -> 11
```

즉 `b`는 그래프를 위아래로 옮기는 기본값이라고 볼 수 있다.

## 딥러닝 모델이 하는 일

모델은 처음부터 좋은 `w`와 `b`를 모른다.

우리가 정답 데이터를 주면, 모델은 여러 번 틀려보면서 좋은 `w`와 `b`를 찾아간다.

```text
처음 생각: w = 아무 값, b = 아무 값
예측하기
정답과 비교하기
손실 계산하기
w와 b 수정하기
반복하기
```

1장에서 배운 리듬과 같다.

```text
입력 -> 예측 -> 손실 -> 수정
```

## 수식을 코드로 바꾸기

수식:

```text
y = wx + b
```

코드:

```python
y = w * x + b
```

수식에서는 `wx`처럼 곱하기 기호를 생략할 수 있지만, 파이썬 코드에서는 반드시 `*`를 써야 한다.

```text
수식: wx
코드: w * x
```

이 차이를 기억하자.

## 예제로 읽어보기

다음 수식을 보자.

```text
y = 3x + 2
```

이 수식은 이렇게 읽는다.

```text
입력값 x에 3을 곱하고 2를 더하면 예측값 y가 된다.
```

`x = 4`라면 다음처럼 계산한다.

```text
y = 3 * 4 + 2
y = 12 + 2
y = 14
```

## 왜 이걸 배워야 하는가

딥러닝이 복잡해 보여도, 아주 작은 기본 단위는 이런 계산에서 출발한다.

나중에 CNN, LSTM, GRU를 배우더라도 모델 안에서는 수많은 가중치와 편향이 조정된다.

지금은 이 정도만 기억하면 된다.

```text
모델은 좋은 w와 b를 찾아가는 기계다.
```

물론 실제 딥러닝 모델에는 `w`와 `b`가 수백 개, 수천 개, 수백만 개까지 있을 수 있다. 하지만 기본 생각은 같다.

## 이번 장에서 반드시 기억할 문장

```text
수식은 외우는 암호가 아니라, 계산 과정을 짧게 적은 문장이다.
```

그리고 이것도 기억하자.

```text
y = wx + b는 입력 x에 가중치 w를 곱하고 편향 b를 더해 예측 y를 만든다는 뜻이다.
```

## 확인 질문

1. `y = wx + b`에서 `x`는 무엇인가?
2. `w`는 무엇인가?
3. `b`는 무엇인가?
4. 수식의 `wx`를 파이썬 코드로 쓰면 어떻게 되는가?
5. 모델이 학습한다는 것은 `w`와 `b`가 어떻게 된다는 뜻인가?
